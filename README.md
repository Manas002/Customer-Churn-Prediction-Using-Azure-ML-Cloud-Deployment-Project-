# Customer-Churn-Prediction-Using-Azure-ML-Cloud-Deployment-Project-

Designed and implemented an end-to-end machine learning pipeline for predicting customer churn using classification algorithms.

Applied feature engineering (tenure grouping, average charge metrics), one-hot encoding, and scaling using ColumnTransformer for robust preprocessing.

Performed hyperparameter tuning using GridSearchCV, improving F1-score and reducing false negatives in churn prediction.

Registered and deployed the trained model using Azure Machine Learning Studio, creating an online inference endpoint.

Enabled real-time prediction via REST API and monitored model performance in a cloud-based environment.
